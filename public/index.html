Enter a site URL (e.g. https://www.hypelocal.com). The tool will:
1) Discover sitemaps
2) Crawl URLs
3) Extract meta title & description
4) Auto-categorize URLs by path
5) Generate a metadata CSV and llms.txt in your requested format.

We’ll first look at /robots.txt for Sitemap: entries. If none are found, we’ll fall back to /sitemap.xml. All HTTP requests go through this app’s backend proxy (no CORS issues).
